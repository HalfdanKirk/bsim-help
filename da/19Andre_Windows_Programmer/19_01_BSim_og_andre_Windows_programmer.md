# BSim og andre Windows programmer

*BSim* er et ægte Windows program, dvs. at alt, hvad der kopieres fra programmets brugergrænseflade, kan indsættes i andre Windows programmer.

Kopieringen sker, som i andre Windows programmer, ved at trykke *Ctrl + c*. Grafik kopieres ved at højre-klikke i den grafiske visning og vælge indgangen *Copy* i menuen hvorved en kopi af det grafiske billede **og** de data som har dannet grafikken kopieres til klipebordet i Windows. Kopien fra klippebordet hentes normalt ind i det ønskede Windows program med *Ctrl + v* eller *Shift + Insert*.

**Tekstbehandlingsprogram**: Er der kopieret grafik, som ønskes indsat i fx MS-Word, skal der benyttes *indsæt speciel*, hvorved typen af indsat materiale kan vælges (fx billede). For en nøjagtig beskrivelse af fremgangsmåden henvises i øvrigt til dokumentationen for de programmer, hvori der ønskes indsat kopieret materiale.

**Regnearksprogram**: Hvis resultaterne ønskes behandlet i et regnearksprogram kan der benyttes *Ctrl + v* eller *Shift + Insert* for indsættelse af data for den kopierede resultatgrafik. Talværdierne indsættes og den videre bearbejdning og grafiske præsentation kan ske i regnearksprogrammet. Der henvises i øvrigt til dokumentationen for de relevante programmer.
